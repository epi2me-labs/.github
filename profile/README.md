# EPI2ME Labs

Welcome to the EPI2ME Labs organization on GitHub!

## About EPI2ME Labs

EPI2ME Labs develops bioinformatics tools and workflows for Oxford Nanopore Technologies sequencing data analysis. Our repositories contain a variety of tools, pipelines, and resources for nanopore sequencing analysis.

## Resources

- **Website**: [epi2me.nanoporetech.com](https://epi2me.nanoporetech.com)
- **Support**: For support inquiries, please visit [nanoporetech.com/about/contact](https://nanoporetech.com/about/contact)

## Getting Started

Browse our repositories to find tools and workflows for:
- Genome assembly and analysis
- RNA sequencing analysis
- Metagenomics
- Structural variant detection
- And more...
