# .github

This is the special `.github` repository for the EPI2ME Labs organization. It contains:

## Contents

- **Community Health Files**: Default files that apply to all repositories in the organization
  - [Code of Conduct](CODE_OF_CONDUCT.md)
  - [Contributing Guidelines](CONTRIBUTING.md)
  - [Security Policy](SECURITY.md)
  - [Support Information](SUPPORT.md)

- **Organization Profile**: The README in `profile/` appears on the organization's GitHub profile page

## Purpose

This repository provides default community health files for all EPI2ME Labs repositories. If a repository doesn't have its own version of these files, GitHub will use the versions from this repository.
