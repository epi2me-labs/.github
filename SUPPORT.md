# Support

Thank you for using EPI2ME Labs tools and workflows!

## Getting Help

### Documentation

First, check if your question is answered in the repository's documentation:

- README files in individual repositories
- Wiki pages (if available)
- Documentation in the `docs/` directory (if available)

### Issues

For bugs, feature requests, or other issues specific to a repository:

1. Search existing issues to see if your question has been asked
2. If not found, open a new issue with detailed information
3. Use issue templates if provided

## Contact Support

These repositories are not proactively monitored for support requests. For more timely help, you may wish to visit:

**Support Contact**: [nanoporetech.com/about/contact](https://nanoporetech.com/about/contact)

## What to Include in Support Requests

When seeking help, please provide:

- Clear description of the issue
- Steps to reproduce the problem
- Expected vs. actual behavior
- System information (OS, version, etc.)
- Relevant log files or error messages
- Version of the tool/workflow you're using

## Response Time

Please note that response times may vary. For urgent issues or commercial support inquiries, please contact Oxford Nanopore Technologies directly through the contact link above.

## Security Issues

For security vulnerabilities, please review our [Security Policy](SECURITY.md) and report issues confidentially.
